### win10 安装路径
#### https://blog.csdn.net/zhicheng_angle/article/details/97623802 
用户\AppData\Roaming\pip\pip.ini  
#### pip.ini文件内容 
[global]  
index-url = https://mirrors.aliyun.com/pypi/simple/  
[install]  
trusted-host=mirrors.aliyun.com
